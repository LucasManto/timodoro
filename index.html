Timodoro

Timodoro is a website that allows you to work using Pomodoro's method. The method is really simple: work 25 minutes, rest 5 minutes. Timodoro will automatically control this workflow by setting alarms.

Click the tomato and start working.

Don't worry about looking how much until rest time, we will alert you :)
